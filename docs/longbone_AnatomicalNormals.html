Function Reference: longbone_AnatomicalNormals

csg-toolkit: [transverse, coronal] = longbone_AnatomicalNormals (v, f)

Compute the normals of the transverse and coronal planes.

This function returns the normals of the transverse and coronal anatomical planes of a longbone mesh defined by its vertices v and faces f.

See also: longbone_AnatomicalPosition, longbone_Registration
